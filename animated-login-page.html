## Animated Login Page

**Type:** Persoonlijk Project

**Jaar:** 2025

**Repo:** https://github.com/N-Chidiac/Animated-login-page

### Opdracht in één zin

Een lichtgewicht, geanimeerde login/registreer-pagina bouwen met pure HTML, CSS en JavaScript, zonder frameworks, om een soepele user experience te demonstreren.

### Wat ik precies moest doen

Dit was geen schoolopdracht, maar een after-school-project om mijn skills in animaties en UI-interactie te oefenen. Ik wou een moderne login-form maken die niet saai is: met een glijdend paneel dat switcht tussen 'Sign In' en 'Sign Up', responsive design, en placeholders voor social logins.

### Mijn aanpak & keuzes

Ik begon met een simpele HTML-structuur: twee forms in een container-div. Met CSS transitions (transform: translateX) maakte ik het paneel gladjes glijden bij toggle – geen zware libraries, gewoon native. JavaScript handelt de klik-events: voeg een 'active'-class toe aan de container, en de rest is pure CSS-magie.

Design: gradient backgrounds voor een modern gevoel, en flexbox voor responsiviteit (werkt perfect op mobiel). Waarom? Omdat ik uit de klantendienst kom, weet ik hoe frustrerend een trage of lelijke login is – dit moest intuïtief en leuk aanvoelen, net als een goed geschilderde muur: strak en zonder krassen.

### Geleerd & trots op

  • CSS-animaties fine-tunen voor smooth transitions.
  • Vanilla JS voor state management (toggle class) zonder overkill.
  • Responsive breakpoints maken die écht werken op alle devices.

Trots op hoe licht het is: < 5KB totaal, laadt instant, en ziet er professioneel uit. Het toont mijn liefde voor code die 'gewoon werkt'

  • een stap van verfkwast naar keyboard.

### Extraatje

Dit is mijn eerste GitHub-repo met een live demo via Pages. Volgende keer voeg ik echte form-validatie en dark mode toe – want waarom niet?

> “Een goede login is als een perfecte eerste laag verf: het zet de toon voor alles wat volgt.”
