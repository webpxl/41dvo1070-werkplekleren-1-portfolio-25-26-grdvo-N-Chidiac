# VS Code Style Portfolio - Nour Chidiac

Dit is mijn persoonlijke portfolio, ontworpen met de look-and-feel van de **Visual Studio Code** editor. Het project is gebouwd als onderdeel van mijn opleiding aan PXL-Hasselt en toont mijn vaardigheden in front-end development door middel van een interactieve interface.

## Features

-   **VS Code Esthetiek:** Volledige editor-ervaring inclusief sidebar, tabs, statusbar en regelnummers.
-   **Steel Grey Light Mode:** Een industrieel, hoog-contrast thema voor optimale leesbaarheid.
-   **Responsive Design:** Geoptimaliseerd voor desktop en mobiel (met hamburger-menu).
-   **Interactieve UI:** Thema-switcher (Light/Dark) en automatische actieve pagina-detectie.
